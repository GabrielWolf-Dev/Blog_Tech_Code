O que é Blog Tech Code?

A Blog Tech Code é um blog estático com o intuito de trazer informações relevantes para fins de curiosidade e estudo sobre o mundo da programação e tecnologia em geral.

Também foi criado como portfólio de Gabriel Wolf, desenvolvedor Front-End. Caso tenha interesse, você pode analizar o código fonte do projeto no Github de Gabriel.

Tecnologias: 💻

  • HTML 5
  • SASS
  • Javascript
